# qdd-cli
A simple CLI for scaffolding Vue.js projects.

## 安装

```bash
npm i qdd-cli -g
```

## Usage

```bash
# help
qdd-cli --help

# create a project
qdd-cli create my-project
```

## prompts

``` json
{
  "prompts": {
    "name": {
      "type": "string",
      "required": true,
      "message": "Project name"
    }
  }
}
```
